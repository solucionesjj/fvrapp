# Modificaciones sugeridas para la aplicación VoterApp

## Objetivo de la nueva funcionalidad

Brindar una interfaz gráfica que le permita al usuario visualizar un mapa y realizar consultas tipo prompt o lenguaje natural donde la IA pueda presentar datos de acuerdo con la información almacenada en la base de datos de votantes registrados y la base de datos entregada por las entidades gubernamentales.

El sistema tendrá consultas predefinidas para mostrar los datos y también filtros previamente definidos.

## Especificaciones gráficas

## Indicadores:

Distribución por Edad (Generacional): Utilizando dateOfBirth, podemos agrupar a los votantes en categorías (Gen Z, Millennials, Boomers). Análisis de distribución de votantes por edad.

Índice de "Safe Drivers": Verificar si esta variable es útil para realizar procesos de analítica.

Análisis de Género: Usando el campo Sex, podemos evaluar la brecha de género en la captura de datos.

Antigüedad de la Licencia: Usando licenseIssueDate con la fecha actual, podemos identificar a los "Nuevos Residentes".

Densidad de los votantes de acuerdo con su código postal: Se podría visualizar la distribución de los votantes.
